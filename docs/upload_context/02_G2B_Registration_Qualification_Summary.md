# 02. 나라장터 등록 품목·자격 요약

## 1. 문서 목적

이 문서는 `YOnLab G2B Agent v2`가 나라장터·공공입찰 공고를 분석할 때 사용할 **자격·품목 적격성 판단 기준**이다.

Agent는 이 문서를 기준으로 다음을 판단한다.

- 와이온랩이 투찰 가능성이 있는 공고인지
- 필수 등록 업종·품목·자격과 와이온랩 기준이 맞는지
- 소기업·소상공인·창업기업 조건이 기회인지
- 실적 제한, 직접생산확인, 지역 제한이 리스크인지
- 추가 확인이 필요한 서류가 무엇인지

이 문서는 내부 기준 문서이며, 실제 입찰 판단 전에는 반드시 최신 공식 증빙자료로 재확인해야 한다.

|---|---|
| 기업명 | 주식회사 와이온랩 |
| 소재지 | 서울특별시 강남구 |
| 기업 규모 | 소기업 / 소상공인 기준 |
| 창업기업 여부 | 초기창업기업 기준 |
| 핵심 업종 | 소프트웨어사업자 |
| 주요 사업 분야 | 패키지소프트웨어 개발·공급, 컴퓨터 관련 서비스, AI 소프트웨어 개발, 클라우드 시스템 구축 |
| 우선 공고 유형 | AI/SW 개발, 정보시스템 구축, 클라우드 플랫폼, AI Agent, 온디바이스 AI 실증, Device Farm, 검증 자동화 |

## 3. 소프트웨어사업자 관련 기준

Agent는 와이온랩을 다음 소프트웨어 사업 분야와 연계하여 판단한다.

| 항목 | 판단 기준 |
|---|---|
| 소프트웨어사업자 일반 현황 관리확인서 | 보유 또는 제출 가능 기준으로 판단 |
| 패키지소프트웨어 개발·공급사업 | AI/SW 제품화, 패키지SW, 플랫폼 개발 공고에 유리 |
| 컴퓨터관련서비스사업 | 정보시스템 개발·구축·운영 공고에 유리 |
| 디지털콘텐츠 개발 서비스사업 | 콘텐츠/플랫폼/서비스형 과업에서 보조 적합성 |
| 데이터베이스 제작 및 검색서비스 | 데이터 수집·검색·분석 시스템 공고에서 보조 적합성 |

### Agent 판단 규칙

- 공고문에 `소프트웨어사업자`, `컴퓨터관련서비스사업`, `패키지소프트웨어개발공급사업`이 명시되면 **적합 가능성 높음**으로 판단한다.
- 공고문에 `1468 소프트웨어사업자` 등 업종코드가 명시될 경우 **최신 경쟁입찰참가자격등록증 확인 필요**로 표시한다.
- SW사업자 확인서, 경쟁입찰참가자격등록증, 사업자등록증, 법인등기부등본의 상호·대표자·주소 일치 여부를 제출 전 확인 대상으로 표시한다.

## 4. 나라장터 등록 품목 기준

와이온랩의 핵심 매칭 품목은 다음으로 판단한다.

| 품목명 | 매칭 수준 | 비고 |
|---|---:|---|
| 인공지능소프트웨어 | 매우 높음 | AI 소프트웨어·AI Agent·AI 분석 시스템 공고 우선 추천 |
| 정보시스템개발서비스 | 매우 높음 | 시스템 구축·웹/API·업무시스템 개발 공고 우선 추천 |
| 패키지소프트웨어개발및도입서비스 | 높음 | 제품화·플랫폼·솔루션 도입형 공고에 적합 |
| 클라우드소프트웨어 | 높음 | 클라우드 대시보드·SaaS·API 시스템에 적합 |
| 시스템관리소프트웨어 | 중간~높음 | 운영·관리·모니터링 시스템에 적합 |
| 화상회의/네트워크회의용 소프트웨어 | 중간 | 협업·원격 제어·원격 검증 UI 공고에서 보조 적합성 |
| 단순 하드웨어·장비 납품 | 낮음 | SW 개발 범위가 없으면 낮은 우선순위 |

### 중요 품목 코드

| 코드 | 품목 | 판단 |
|---|---|---|
| 4323269801 | 인공지능소프트웨어 | 와이온랩 핵심 매칭 품목 |
| 8111159901 | 정보시스템개발서비스 | 정보시스템 개발·구축 공고 핵심 검토 대상 |

> 주의: 품목 코드와 등록 상태는 실제 입찰 전 최신 나라장터 경쟁입찰참가자격등록증으로 확인해야 한다.

## 5. 소기업·소상공인·창업기업 기준

| 조건 | 와이온랩 판단 | 추천 가중치 |
|---|---|---:|
| 소기업 제한 | 유리 | 높음 |
| 소상공인 제한 | 매우 유리 | 매우 높음 |
| 중소기업자 간 경쟁 | 가능성 있음 | 중간~높음 |
| 창업기업 확인서 우대 | 매우 유리 | 매우 높음 |
| 여성기업·장애인기업 등 특정 우대 | 해당 여부 별도 확인 | 보류 |
| 대기업 참여 제한 | 간접적으로 유리 | 중간 |

### Agent 판단 규칙

- `소기업`, `소상공인`, `창업기업`, `창업기업 확인서`가 공고문에 있으면 추천 점수를 가산한다.
- 제출 필요 서류에 다음을 표시한다.
  - 소기업·소상공인 확인서
  - 창업기업 확인서
  - 소프트웨어사업자 일반 현황 관리확인서
  - 경쟁입찰참가자격등록증
- 창업기업 우대는 단순 가점이 아니라 **전략적 우위**로 리포트에 표시한다.

## 6. 지역 제한 기준

와이온랩의 소재지는 **서울특별시 강남구**로 판단한다.

| 공고 지역 조건 | Agent 판단 |
|---|---|
| 전국 | 적합 |
| 서울특별시 | 적합 |
| 서울 강남구 | 매우 적합 |
| 수도권 | 대체로 적합 |
| 경기도·인천 등 특정 지역 제한 | 실제 조건 확인 필요 |
| 대전·부산·광주 등 타 지역 제한 | 리스크 높음 |
| 발주처 방문·상주 필수 | 수행 리스크로 표시 |

### Agent 판단 규칙

- `서울`, `전국` 조건은 추천 가능성을 높인다.
- 서울 외 지역 제한은 자동 탈락으로 단정하지 말고, **지역 제한 리스크**로 표시한다.
- 지역 제한이 명확히 `해당 지역 소재 업체`로 되어 있으면 Pass/Fail에서 Fail 가능성을 표시한다.

## 7. 실적 제한 기준

와이온랩은 초기창업기업이므로 강한 실적 제한은 주요 리스크다.

| 실적 제한 문구 | 판단 |
|---|---|
| 최근 3년 단일 계약 1억 이상 | 리스크 높음 |
| 최근 3년 유사 사업 수행 실적 | 증빙 확인 필요 |
| 공공기관 수행 실적 필수 | 리스크 높음 |
| 단일 건 5천만 원 이상 | 보유 실적 증빙 여부 확인 |
| 민간 실적 인정 | 메인플라츠 실적 활용 가능성 검토 |
| 실적 제한 없음 | 초기기업에 유리 |

### Agent 리포트 문구 예시

> 본 공고는 최근 3년 유사 사업 실적 제출을 요구할 가능성이 있어, 와이온랩의 신규 법인 특성상 공식 실적증명서·계약서·세금계산서·검수확인서 확보 여부를 먼저 확인해야 합니다.

## 8. 직접생산확인 기준

정보시스템 개발서비스 등 일부 공고는 직접생산확인증명서를 요구할 수 있다.

| 조건 | 판단 |
|---|---|
| 정보시스템개발서비스 직접생산확인 필수 | 제출 가능 여부 확인 필요 |
| 중소기업자 간 경쟁제품 | 직접생산확인 여부 중점 검토 |
| 단순 소액 수의계약 | 요구 여부 공고별 확인 |
| SW 개발용역이나 직접생산 미기재 | 일반 SW사업자 요건 중심 검토 |

### Agent 판단 규칙

- 공고문에 `직접생산확인`, `직생`, `중소기업자간 경쟁제품`이 있으면 리스크로 표시한다.
- `8111159901 정보시스템개발서비스`와 함께 등장하면 **제출 필수 가능성 높음**으로 표시한다.
- 확인서 보유 여부가 불명확하면 `확인 필요`로 표시한다.

## 11. Agent Pass/Fail 판단 규칙

### Pass 가능성이 높은 경우

- AI/SW/정보시스템/클라우드 관련 공고
- 소기업·소상공인 대상 공고
- 창업기업 우대 또는 창업기업 대상 공고
- 서울 또는 전국 지역 조건
- 실적 제한이 없거나 낮은 공고
- 와이온랩 등록 품목과 직접 일치하는 공고

### Fail 또는 고위험 가능성이 높은 경우

- 특정 지역 소재 업체만 참여 가능하고 서울이 아닌 경우
- 최근 3년 공공기관 대형 실적 필수
- 특정 제조사 총판·공식 대리점 자격 필수
- 직접생산확인 필수이나 보유 여부 불명확
- 단순 하드웨어 납품 중심
- 상주 인력 다수·장기 운영 필수
