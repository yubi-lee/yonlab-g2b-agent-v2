# 05. 공고 추천 리포트 예시

## 1. 문서 목적

이 문서는 `YOnLab G2B Agent v2`가 공고 분석 후 생성해야 하는 **최종 추천 리포트 형식**을 고정하기 위한 예시 문서다.

Agent와 개발자는 이 문서를 기준으로 다음을 구현한다.

- 추천 리포트 Markdown 템플릿
- 점수·추천 사유·리스크 표시 방식
- 제출 서류 체크리스트
- 입찰 준비 전략 문구
- 와이온랩 맞춤형 공고 분석 결과 형식

## 3. 추천 등급 기준

| 점수 | 등급 | 판단 |
|---:|---|---|
| 85~100 | 적극 추천 | 와이온랩 자격·기술·전략과 매우 부합 |
| 70~84 | 추천 | 검토 가치 높음 |
| 55~69 | 조건부 추천 | 리스크 확인 후 진행 |
| 40~54 | 보류 | 공고 조건 또는 실적 리스크 큼 |
| 0~39 | 비추천 | 자격·기술·지역·실적 조건이 부적합 |

## 5. 샘플 추천 리포트

## 🎯 와이온랩 맞춤 추천 공고: AI 기반 행정지원 업무 자동화 시스템 구축

- **매칭 점수**: 86점 / 100점
- **추천 등급**: 적극 추천
- **추천 사유**:
  - 공고의 핵심 과업이 AI 기반 업무 자동화 및 정보시스템 구축으로, 와이온랩의 AI 소프트웨어·AI Agent 개발 방향과 직접 부합합니다.
  - `소프트웨어사업자` 및 `소기업 또는 소상공인` 조건이 명시되어 있어 와이온랩의 현재 조달 진입 전략과 적합성이 높습니다.
  - `창업기업 우대` 조건이 포함되어 있어 초기창업기업인 와이온랩에 전략적으로 유리합니다.
  - 서울특별시 지역 조건은 와이온랩의 소재지인 서울특별시 강남구와 부합합니다.
  - 클라우드 기반 시스템 구축, AI 업무 자동화, 인공지능소프트웨어 키워드가 모두 포함되어 기술 적합성이 높습니다.

- **핵심 정보**:
  - 발주처: 서울특별시 산하기관
  - 예산: 55,000,000원
  - 마감일: 2026-07-15
  - 계약 방식: 협상에 의한 계약
  - 지역 조건: 서울특별시

- **와이온랩 적합성 판단**:
  - SW사업자 조건: 적합 가능성 높음
  - 소기업·소상공인 조건: 유리
  - 창업기업 조건: 우대 가능성 높음
  - 등록 품목 적합성: 인공지능소프트웨어, 정보시스템개발서비스와 부합
  - 기술 적합성: AI Agent, 클라우드 시스템, 업무 자동화 역량 활용 가능

- **입찰 준비 전략**:
  - 예비창업패키지 기반의 온디바이스 AI 검증 플랫폼 및 Device Farm 구축 경험을 “AI 시스템의 실환경 검증 역량”으로 연결해 제안서에 반영합니다.
  - 단순 AI 기능 개발이 아니라, 운영 가능한 AI 업무 자동화 시스템을 설계·검증·배포할 수 있다는 점을 강조합니다.
  - 와이온랩의 AI/SW 원격 검증 구조를 활용하여 산출물의 품질 검증, 테스트 자동화, 운영 안정성 확보 방안을 제시합니다.
  - 메인플라츠 플랫폼 구축·납품 경험은 민간 플랫폼 구축 경험으로 활용 가능하되, 실적 제출 가능 범위와 증빙자료를 사전에 확인해야 합니다.

- **제출 필요 서류**:
  - 경쟁입찰참가자격등록증
  - 소프트웨어사업자 일반 현황 관리확인서
  - 소기업·소상공인 확인서
  - 창업기업 확인서
  - 사업자등록증
  - 법인등기부등본
  - 신용평가등급확인서
  - 유사 사업 실적증명서 또는 계약·세금계산서·검수확인서

- **리스크**:
  - 최근 3년 유사 사업 수행 실적 제출 조건이 실제 필수 요건인지 확인해야 합니다.
  - 공공기관 실적만 인정하는 경우 와이온랩의 신규 법인 특성상 감점 또는 부적격 가능성이 있습니다.
  - 협상에 의한 계약이므로 제안서 완성도, 발표 대응, 인력 투입 계획의 설득력이 중요합니다.

- **권장 대응**:
  - 공고문 원문에서 실적 조건이 “필수”인지 “평가 요소”인지 먼저 확인합니다.
  - 민간 실적 인정 여부를 확인하고, 메인플라츠 플랫폼 구축 관련 증빙자료를 사전 정리합니다.
  - 제안서에는 AI Agent 기능보다 공공기관 업무 개선 효과, 운영 안정성, 검증 체계, 유지보수 계획을 균형 있게 반영합니다.
  - 공고 마감 전 소기업·소상공인 확인서와 창업기업 확인서의 유효기간을 확인합니다.

## 6. 조건부 추천 예시

## 🎯 와이온랩 맞춤 추천 공고: 클라우드 기반 데이터 통합관리 플랫폼 구축

- **매칭 점수**: 68점 / 100점
- **추천 등급**: 조건부 추천
- **추천 사유**:
  - 클라우드 플랫폼 및 정보시스템 개발 과업은 와이온랩의 SW 개발 역량과 부합합니다.
  - 데이터 통합관리 시스템은 AI Agent 및 공공 데이터 분석 서비스로 확장 가능성이 있습니다.
  - 다만 AI/온디바이스/Device Farm과의 직접 연관성은 낮아 차별화 포인트가 제한적입니다.

- **핵심 정보**:
  - 발주처: 00공공기관
  - 예산: 120,000,000원
  - 마감일: 2026-08-01
  - 계약 방식: 제한경쟁
  - 지역 조건: 전국

- **와이온랩 적합성 판단**:
  - SW사업자 조건: 적합 가능성 있음
  - 소기업·소상공인 조건: 공고문 확인 필요
  - 창업기업 조건: 우대 여부 확인 필요
  - 등록 품목 적합성: 정보시스템개발서비스와 부합
  - 기술 적합성: 클라우드 시스템, 데이터 관리, API 플랫폼 역량 활용 가능

- **입찰 준비 전략**:
  - 와이온랩의 AI Agent 기반 공공 데이터 분석 확장 가능성을 부가 제안으로 제시합니다.
  - 단순 관리 시스템이 아니라, 향후 AI 분석·자동 리포트·업무 자동화로 확장 가능한 구조를 강조합니다.
  - 프로젝트 규모가 큰 경우 컨소시엄 또는 협력사 투입 가능성을 검토합니다.

- **제출 필요 서류**:
  - 경쟁입찰참가자격등록증
  - 소프트웨어사업자 일반 현황 관리확인서
  - 사업자등록증
  - 법인등기부등본
  - 신용평가등급확인서
  - 실적증명서

- **리스크**:
  - 예산 규모가 초기기업 단독 수행에는 다소 클 수 있습니다.
  - 최근 3년 유사 실적 또는 전담 인력 조건이 있을 경우 리스크가 있습니다.
  - AI 핵심 기술 공고가 아니므로 와이온랩의 차별성이 약해질 수 있습니다.

- **권장 대응**:
  - 실적 제한과 상주 인력 조건을 우선 확인합니다.
  - 단독 입찰보다 협력사 구성 가능성을 검토합니다.
  - AI 확장 기능을 선택 제안으로 넣되, 본 과업의 기본 요구사항을 우선 충족하는 방향으로 제안합니다.

## 7. 비추천 예시

## 🎯 와이온랩 맞춤 추천 공고: 사무용 전산장비 일괄 구매

- **매칭 점수**: 28점 / 100점
- **추천 등급**: 비추천
- **추천 사유**:
  - 과업의 중심이 사무용 전산장비 구매와 납품으로, 와이온랩의 AI/SW 개발 역량과 직접 부합하지 않습니다.
  - 인공지능소프트웨어, 정보시스템개발서비스, 클라우드소프트웨어 등 핵심 등록 품목과의 연관성이 낮습니다.
  - 특정 제조사 납품 또는 대리점 자격이 요구될 가능성이 있어 기술기업으로서의 경쟁 우위가 제한적입니다.

- **핵심 정보**:
  - 발주처: 00기관
  - 예산: 30,000,000원
  - 마감일: 2026-07-20
  - 계약 방식: 물품 구매
  - 지역 조건: 전국

- **와이온랩 적합성 판단**:
  - SW사업자 조건: 직접 관련 낮음
  - 소기업·소상공인 조건: 확인 필요
  - 창업기업 조건: 우대 효과 제한적
  - 등록 품목 적합성: 낮음
  - 기술 적합성: 낮음

- **입찰 준비 전략**:
  - 본 공고는 와이온랩의 핵심 기술력을 보여주기 어렵기 때문에 우선순위를 낮춥니다.
  - 장비 납품 이후 AI/SW 구축 과업이 별도로 존재하는지 확인하는 정도로 제한합니다.

- **제출 필요 서류**:
  - 경쟁입찰참가자격등록증
  - 사업자등록증
  - 납품 관련 제조사 또는 대리점 증빙이 요구될 수 있음

- **리스크**:
  - 단순 물품 납품 중심으로 수익성과 전략성이 낮습니다.
  - 제조사 공급확약서 또는 대리점 자격이 필요할 수 있습니다.
  - 와이온랩의 AI/Device Farm/클라우드 역량을 활용하기 어렵습니다.

- **권장 대응**:
  - 직접 입찰은 권장하지 않습니다.
  - 유사 발주처의 후속 AI/SW 구축 공고를 모니터링합니다.

- “소기업·소상공인 제한 공고로 초기기업인 와이온랩에 유리합니다.”

- “무조건 수주 가능”
- “자격 요건 충족 확정”
- “실적 제한 문제 없음”
- “공고문 확인 없이 참여 가능”
- “최저가로 쉽게 수주 가능”

## 9. 리스크 문구 규칙

리스크는 과장하지 말고 구체적으로 표시한다.

| 리스크 유형 | 문구 예시 |
|---|---|
| 실적 제한 | 최근 3년 유사 사업 실적 요건이 필수일 경우, 신규 법인 특성상 증빙 리스크가 있습니다. |
| 지역 제한 | 서울 외 특정 지역 소재 업체 제한이 있을 경우 투찰 자격을 확인해야 합니다. |
| 직접생산확인 | 정보시스템개발서비스 직접생산확인증명서 요구 여부를 확인해야 합니다. |
| 상주 인력 | 상주 또는 전담 인력 다수 투입 조건이 있을 경우 수행 부담이 있습니다. |
| 특정 제조사 | 제조사 공급확약서 또는 대리점 자격이 요구될 경우 와이온랩 단독 참여가 어려울 수 있습니다. |

## 12. 결론

추천 리포트는 단순 요약문이 아니라, 와이온랩이 실제로 입찰 여부를 판단하기 위한 **의사결정 문서**다.

따라서 모든 리포트는 다음 세 가지를 분명히 해야 한다.

```text
1. 왜 추천하는가
2. 무엇을 준비해야 하는가
3. 어떤 리스크가 있는가
```
